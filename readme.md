React.memo(<компонент>, (prevProps) => false) 
- оборачивает и не перерисовывает компонент если родитель перерисовывается и нет входящих пропсов,
- если есть входящие пропсы можно отменить перерисовку вернув false в функции

React.useMemo(тяжело-вычисляемая функция, [зависимости])

- запоминает вычисления, и делает перерасчет если зависимости поменялись
- может оборачивать и не перерисовывать компонент если нет зависимостей
